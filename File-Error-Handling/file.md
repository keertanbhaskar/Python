# File handling
- file handling refers to process of performing operations on a file, such as creating, opening, reading, writing and closing.

# Why we need file handling
## Solves the problem:

- data saves temporarily, file handling helps to save the data permanently
- when the program closes, everything is lost, stores data even after the program ends
